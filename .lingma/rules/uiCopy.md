所有 UI 文件内存在的“状态栏（时间、信号、WiFi、电量）”和“顶部导航栏（含标题栏）”都是为了模拟 iOS 效果，不属于业务页面内容。

生成微信小程序页面时必须遵守以下规则：
1. UI 中的状态栏和顶部导航栏一律忽略，不要出现在 WXML / WXSS 中。
2. 微信小程序页面必须使用原生 navigationBar（由 page.json 或 app.json 控制）。
3. 页面中真正需要生成的部分是“顶部导航栏下面的所有内容区域”，其他全部忽略。
4. 若 UI 顶部区域有实际业务内容（例如 banner、顶部统计栏），仅保留业务内容，不保留模拟导航栏部分。
5. 任何情况下禁止在页面内重复生成自定义顶部导航栏（除非映射表中明确说明需要自定义导航栏）。
6. 底部 TabBar 也由 app.json 管理，不从 UI 文件中生成。
